This project is a Learning Analytics Dashboard developed using React JS (JSX) and Recharts. It visualizes learner enrollment, performance, assessment completion, and district-wise rankings. The dashboard supports Monthly and Quarterly views, Light and Dark themes, and is fully responsive. Data is loaded from the provided JSON files and displayed using interactive charts as per the given UI mockups.
